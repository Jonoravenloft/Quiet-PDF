# Quiet PDF

A calm, **private, on-device** PDF markup editor for Windows. Open any PDF, draw on
it, add text and checkmarks, drop your saved signature, redact sensitive areas,
select and copy text, then save or print — all completely offline.

**Nothing is ever uploaded.** Every PDF is opened, edited, and saved right on your own
computer. Nothing you do leaves your machine, and it works with no internet connection.

---

## Download

Get the latest version from the [**Releases**](../../releases/latest) page:

- **`Quiet-PDF-Setup-x.y.z.exe`** — installs the app with a Start-menu shortcut.
- **`Quiet-PDF-Portable-x.y.z.exe`** — runs without installing (handy from a USB stick).

### First-run warning (this is normal)
Quiet PDF is made by one person, not a big signed software company, so Windows may show
**“Windows protected your PC.”** Click **More info → Run anyway**. As more people
download it over time, that warning goes away on its own.

## Verify your download (recommended)

Every release lists a **SHA-256 checksum** for each file — a unique fingerprint you can
use to confirm your download is the real, untampered file. Open PowerShell in your
Downloads folder and run:

```
Get-FileHash .\Quiet-PDF-Setup-x.y.z.exe -Algorithm SHA256
```

Compare the result to the value in that release’s notes. If they match exactly, your
copy is identical to the one published here. If they don’t, delete it and download again.

## What it does

- Open regular and form PDFs
- Draw freehand, highlight, and add text boxes
- Checkmark and X stamps
- Place a saved signature and initials
- Redact sensitive areas
- Select and copy the PDF’s text
- Save an edited PDF (including a **flattened** mode that truly removes redactions), or print

---

## Privacy

No accounts, no tracking, no network features. Quiet PDF never sends your files anywhere.
